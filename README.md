# tool-depot-supplier-dashboard

Free Streamlit MVP for supplier selection and shipping cost analysis.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Features

- Editable products, suppliers, and carrier rates
- Hammer supplier A vs B scenario comparison
- Growth %, hammer cost, hammer weight, and truck capacity controls
- Weekly cost detail plus annual supplier summary
- CSV downloads for results
